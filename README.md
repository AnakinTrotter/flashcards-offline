# Flashcards Offline
To create a flashcard set:\
Make a text document (notepad is good) with the following format:\
Front| Back\
Front|Back\
Front|Back\
etc...

Controls:\
Left/Right arrow keys to cycle cards\
Up/Down/Spacebar to flip cards
s to shuffle cards
